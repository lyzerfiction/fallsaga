FAQ

Is the game free?

The demo version of Fallsaga is completely free and includes only Chapter 1. The full game will be a paid release once it's available.

Which platforms is the game available on?

The demo is currently available only on PC. After the full release, we might bring it to other platforms as well — who knows?

Will the full version also be released on Gamejolt?

No. We plan to release the full version on Steam first. A Steam page will be launched when the game is close to completion. As of now, we have no plans to release the paid version on any platform other than Steam for PC.

I can't save while playing the demo. Why is this happening?

Unfortunately, this is a known issue. The solution is actually quite simple: make sure you place the demo files in a folder that doesn't require administrator permission. When the game tries to save, it needs to create a new file — but if the folder requires admin rights, the game won’t be able to do that.

I found some bugs in the game. How can I report them?

You can make a big difference by reporting any bugs you find! Just send the details to mete.ylmz180@gmail.com — your feedback helps us improve the game.

What is Fallsaga?

Fallsaga is a story-driven game that follows Marcel, a young man searching for a legendary waterfall. Along the way, Marcel faces various challenges and stumbles into bizarre events. The game features a rich narrative, memorable characters, plenty of dialogue, and thought-provoking puzzles. You'll also encounter occasional gameplay events with fun — or sometimes stressful — mechanics (depending on how you see it :d).

How can I follow the development of the game?

You can keep up with the game's development by visiting the DEVLOG page on our website. We'll be posting important updates and progress there.
