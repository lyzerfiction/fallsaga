FAQ

Is the game free?

The demo of Fallsaga is completely free. The demo consists of Chapter 1 only. The game will be paid after the full version is released.

Which platforms is the game available on?

The demo is currently only available on the PC platform. Once the full version is released, we may release it for other platforms. Who knows?

Will the full version also be released on Gamejolt?

No. We plan to release the full version on Steam first. We will open a Steam page when the game is close to completion. We have no plans to release the game for a fee on any platform other than Steam on the PC platform. That's the status for now.

I can't save while playing the demo. Why is this happening?

Unfortunately, we are faced with such a problem. Actually, the solution is simple. When you download the demo, you need to store the files in a location that does not require administrator permission. Because the game creates a new file when saving and this new file cannot be created because it requires administrator permission.

I found some bugs in the game, how can I report it?

You can make a huge contribution to fixing these bugs by sending the bugs you find to mete.ylmz180@gmail.com.

What is Fallsaga?

Fallsaga is a story-based game about a guy named Marcel searching for a waterfall of legend. Throughout the game, Marcel has to deal with different challenges and encounters absurd events. The game has a good story, interesting characters, lots of dialog and puzzles. There are also occasional events with fun (or stressful, it's up to you :d) mechanics.

How can I follow the development of the game?

You can follow the development of the game on the DEVLOG page on our website. We will share important developments on that page.

I want to contribute to the development of the game. How can I do that?

You can create a pre-order registration. If you want to contribute more, you can contact me via e-mail.

How can I get details about creating a pre-order registration?

You can find all the relevant details on the PRE-ORDER page on our website.
